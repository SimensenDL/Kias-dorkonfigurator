## 1. Generelle regler

### Fellestrekk for alle KIAS-dører

- **Vedlikeholdsfrie materialer** beregnet for daglig renhold
- Godt egnet i miljøer med høy fuktighet
- Standard grunnfarge: **RAL 9010 (hvit)**
- Kan lakkeres i valgfri **RAL Classic** eller **NCS**-farge på forespørsel
- Kan leveres med glassfelt
- Alle mål i **millimeter (mm)**
- Slagretning angis som **V** (venstre) eller **H** (høyre)

### Materialer (fellesnevner)

| Komponent | Standard materiale |
|---|---|
| Karm | Pulverlakkert aluminium |
| Tetningslist | O-pakning / dempelist i silikon (montert på karm) |
| Dørramme | Pulverlakkert aluminium |
| Overflate dørblad | Glassfiberarmert polyester (GRP) med gelcoat yttersjikt |
| Isolasjon | Skummet polyuretan (PUR), KFK-fri, 45 kg/m³ |
| Kantforsterkning | Pulverlakkert aluminium |
| Slepelist | Silikon |

### Viktige merknader

- Ved ettermontering av dørlukker eller annet utstyr på dørbladet, **må medsendte blindmutrer benyttes**
- Ytterdører (YD) er **ikke testa og godkjente iht. gjeldende krav til ytterdører**, ref. NS-EN 14351-1
- 2-fløyede dører leveres med karm **ikke ferdig sammensatt**

## 3. Dørblad - Spesifikasjoner

### Standardtykkelser per dørtype

| Tykkelse | Dørtyper |
|----------|---------|
| **40 mm** | SDI (innerdør), PDI (isolert pendel), BR (brakke), FDI (fjøs inner), skyvedør (standard) |
| **52 mm** | BD (branndør) |
| **60 mm** | KD (kjølerom), BO (bod), FD (fjøs ytter), YD (ytterdør), BA (badstu), skyvedør (kjøle) |
| **5 mm** | PDPC (polykarbonat klar/sotet), PDPO (polykarbonat opalhvit) |

### Dørblad konstruksjon

#### Isolerte dørblad (GRP)

- Overflate: Glassfiberarmert polyester (GRP) med gelcoat yttersjikt
- Kantforsterkning: Pulverlakkert aluminium
- Isolasjon: Skummet polyuretan (PUR), 45 kg/m³
- Standard farge: Hvit (RAL 9010)
- 30 års garanti mot fukt- og råteskader

#### Polykarbonat dørblad (pendeldører)

- Materiale: 5 mm polykarbonat (klar, sotet eller opalhvit)
- Dørramme: Pulverlakkert eller natureloksert aluminium
- Sparkeplate: Narvet polykarbonat
- Slepelist: Silikon
- Ryggforsterkning: Natureloksert aluminium

#### Branndørblad

- Isolasjon: **Rockwool (steinull)** — ikke polyuretan
- Brannmotstand: B30 (30 minutter)

#### Kjøleromsdørblad

- **Brutt kuldebro** i dørrammen
- Tilgjengelig i 40 mm og 60 mm (standard kjøledør: 60 mm)

### Pendeldør-tilleggsutstyr

| Komponent | Beskrivelse |
|-----------|-------------|
| Vindu | 300 x 400 mm i 5 mm polykarbonat |
| Sparkeplate | 1 mm narvet polykarbonat (nedre halvpart, inn-/utside) |
| Slepelist | PVC |
| Klemsikring | I bakkant av dørbladene |
| Avviserbøyler | Tilleggsutstyr |
| Demonterbare sliteplater | 5 mm PEHD, H=500 mm (4 stk, for port) |
